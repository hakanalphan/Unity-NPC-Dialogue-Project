 Unity NPC Dialogue Project

Bu proje, Unity kullanarak oluşturulmuş bir NPC etkileşim ve diyalog sistemidir.
Oyuncu, sahnedeki NPC'lere yaklaşarak E tuşu ile etkileşime girer, ardından ekranda bir diyalog paneli açılır. NPC’ler arası farklı konuşmalar

 Özellikler

 Çok satırlı diyalog sistemi (DEVAM / KAPAT butonları ile kontrol)


 Trigger bazlı etkileşim mesafesi kontrolü


 Projeyi Çalıştırma

Bu repo bir Unity projesidir. Açmak için:

1. Unity Hub’ı aç

Add → proje klasörünü seç → Open.

2. Doğru sürümü kullan (önerilen)

Unity 2022.3 LTS

3. Firebase Kullanıyorsan

Assets/StreamingAssets klasörüne kendi Firebase projenin
google-services.json dosyasını eklemeyi unutma.

4. Sahne Sırası

Build Settings’te sahnelerin sırası şu şekilde olmalıdır:

0  LoginScene
1  GameScene
